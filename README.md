# Bhahi-khata Backend

## About

Bhahi-khata is an expense manager helps to make your expense count. Helps to automate the rediculous task of managing and keeping track of the monthly and yearly expenses. 


## Tech-stack used

- Node.js 
- Express.js
- MongoDB 
- Firebase
